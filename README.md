# Designing Data-Intensive Applications — 학습 정리

Martin Kleppmann의 *Designing Data-Intensive Applications* (DDIA)를 챕터별로 정리하고
퀴즈로 복습하기 위한 학습 프로젝트입니다.

## 정리 원칙

- 설명은 한국어로, **중요 기술 용어는 영어 원문 그대로** 유지 (예: replication, partitioning, consensus)
- 각 챕터마다 핵심 개념을 상세히 + 쉬운 비유/예시로 설명
- 각 챕터 끝에 **약 20문제 퀴즈**(객관식 + 단답형 + O/X) + 정답/해설

## 각 챕터 노트 구성

1. 챕터 개요
2. 핵심 개념 정리
3. 쉬운 예시 / 비유
4. 핵심 용어 정리 (Glossary)
5. 한 줄 요약
6. 퀴즈 (약 20문제)
7. 정답 & 해설
